# Аналитик поэзии журнала «Юность»

Приложение для визуализации и анализа поэтического корпуса журнала **«Юность»** за **1955–1974 годы**.

## Ссылка на приложение

[Открыть приложение на shinyapps.io](https://egorvoron.shinyapps.io/yunost-shiny/)

## Возможности

- Динамика поэзии по номерам за 1955–1974 годы
- Облако слов поэтических произведений
- Лента времени сотрудничества поэтов с журналом
- Граф совместных публикаций с настраиваемыми параметрами
